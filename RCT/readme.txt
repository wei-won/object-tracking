=================Contents====================
- no_pretrain_v1.2 (reservoir as input of softmax) 
  The modified visual tracker (using Echo State Network as classifier). The activations of the reservoir are fed to softmax for classification. 

- no_pretrain_v2.0(readout as input of softmax) 
  The modified visual tracker (using Echo State Network as classifier). The outputs of the readout layer are fed to softmax for classification. 

- Data
  Stores the videos used.


Note: For more details of the two versions, please refer to the readme.txt files within each folder.
